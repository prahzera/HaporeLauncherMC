HAPORELAUNCHER v2.0.0
======================

¡Bienvenido a HaporeLauncher!

HaporeLauncher es un launcher de Minecraft moderno y fácil de usar que te permite
jugar tanto con cuentas premium como no-premium.

CARACTERÍSTICAS PRINCIPALES:
---------------------------
• Interfaz moderna con diseño glassmorphism
• Soporte para cuentas premium y no-premium
• Gestión de múltiples perfiles de juego
• Descarga automática de versiones de Minecraft
• Detección y descarga automática de Java
• Soporte para modloaders (Forge, Fabric, Vanilla)
• Logs en tiempo real durante el lanzamiento
• Editor de perfiles intuitivo
• Temas personalizables

REQUISITOS DEL SISTEMA:
----------------------
• Windows 10 o superior (64-bit)
• Conexión a Internet para descargar versiones
• Mínimo 4GB RAM (recomendado 8GB+)
• Espacio en disco: 2GB mínimo

INSTALACIÓN:
------------
1. Ejecuta el instalador como administrador
2. Sigue las instrucciones del asistente
3. El launcher se instalará en el directorio seleccionado
4. Se crearán accesos directos en el escritorio y menú inicio

PRIMER USO:
-----------
1. Abre HaporeLauncher desde el menú inicio o escritorio
2. Crea tu primer perfil de juego
3. Selecciona una versión de Minecraft
4. Configura tu nombre de usuario
5. ¡Haz clic en "¡JUGAR!" y disfruta!

DIRECTORIOS IMPORTANTES:
------------------------
• Datos del usuario: %LOCALAPPDATA%\HaporeLauncher\
• Versiones de Minecraft: %LOCALAPPDATA%\HaporeLauncher\versions\
• Java descargado: %LOCALAPPDATA%\HaporeLauncher\java\
• Instancias del juego: %LOCALAPPDATA%\HaporeLauncher\instances\

SOPORTE:
--------
Si encuentras algún problema o tienes preguntas:
• Revisa los logs en la interfaz del launcher
• Verifica que tienes una conexión a Internet estable
• Asegúrate de que tu antivirus no bloquee el launcher
• Comprueba que tienes permisos de administrador si es necesario

NOTAS IMPORTANTES:
------------------
• El launcher descarga automáticamente las versiones de Minecraft que necesites
• Java se descarga automáticamente si no está disponible en tu sistema
• Los perfiles y configuraciones se guardan automáticamente
• El launcher permanece abierto durante el juego para monitorear el proceso
